# eaze-js
Time (t) based JS easing functions (cubic, quad, quint, circ...)

## usage

````javascript
  // t valur must be in the [0-1] range
  // result valuest in the [0-1] range too
  var r1 = eaze.in.quad(t);
  var r2 = eaze.out.quad(t);
  var r3 = eaze.in_out.quad(t);
  //...
````
